# emirocak.github.io

GitHub Pages için hazırlanmış kişisel iOS uygulama ve oyun vitrini.

## Yayına Alma

1. Bu dosyaları `emirocak.github.io` adlı GitHub reposuna koyun.
2. `app-ads.txt` içindeki `pub-0000000000000000` değerini kendi AdMob publisher ID'nizle değiştirin.
3. GitHub Pages açık olduğunda site şu adreste çalışır: `https://emirocak.github.io`

## Dosyalar

- `index.html`: Ana sayfa ve uygulama vitrini
- `privacy.html`: Genel gizlilik politikası
- `app-ads.txt`: Reklam ağı doğrulama dosyası
- `apps/*.html`: Uygulama ve oyun detay sayfaları
